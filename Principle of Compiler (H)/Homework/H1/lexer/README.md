# Lexer词法分析程序
## 目录文件说明
--relop.h 分析程序类头文件   
--relop.cpp 分析程序类主文件   
--test.cpp 测试程序主文件   
--test.txt 测试内容文件   
--Makefile 交叉编译   
--answer.md 解题思路文件   
--readme.md 使用说明文件
## 使用方式
1.编译：直接在lexer根目录下运行make指令即可完成编译。   
2.编写测试代码：在test.txt文件中可编写自需的测试序列（原test中已提供一个）。   
3.运行test执行文件，在命令行中直接得到结果。
## 生成结果说明
对于本次测试中生成的结果，形如：   
relop:<= position:from 8 to 9   
表示词法识别出一个<=符号，且该符号位于原字符流中的第8-9位（此处流从第一位开始）。
